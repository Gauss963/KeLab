About

A lab website shaped for clarity, recruiting, and research visibility.

This starter site follows the information-first structure common to academic lab pages: clear identity, visible people, research themes, current updates, and easy contact paths. The layout is intentionally lightweight so it can be hosted directly on GitHub Pages without any build step.
