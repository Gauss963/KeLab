# KeLab Website

Static lab website starter designed for GitHub Pages.

## Files

- `index.html`: Main page content and section structure
- `styles.css`: Layout, responsive design, and visual styling
- `script.js`: Member cards, mobile navigation, and reveal animations

## GitHub Pages Deployment

1. Push this folder to a GitHub repository.
2. Open the repository settings on GitHub.
3. Go to `Pages`.
4. Set the source to deploy from the main branch root.
5. Save and wait for the site URL to be generated.
